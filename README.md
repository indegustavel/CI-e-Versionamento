# CI e Versionamento

[![Meu Primeiro CI](https://github.com/indegustavel/CI-e-Versionamento/actions/workflows/ci.yml/badge.svg)](https://github.com/indegustavel/CI-e-Versionamento/actions/workflows/ci.yml)

Repositório simples para estudos de integração contínua e versionamento.

## Conteúdo

- `app.py`: contém a função `somar(a, b)`.
- `test_app.py`: contém um teste automatizado para a função.
- `requirements.txt`: lista a dependência do projeto.

## Requisitos

- Python 3
- `pytest`

## Como executar os testes

1. Instale as dependências:
   `pip install -r requirements.txt`
2. Execute os testes:
   `pytest`


## Objetivo

Servir como exemplo básico para praticar versionamento com Git e execução de testes automatizados.
